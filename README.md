# Schrifterkennungs-KI

Erkennt handgeschriebene Buchstaben mit einem CNN-Modell und einer GUI.

---

## Installation

1. Python 3.10+ installieren  
2. Dateien herunterladen:
   Repository klonen und sicherstellen, dass folgende Dateien im selben Ordner liegen: 
   - `Letters_A-M.zip`
   - `Letters_N-Z.zip`
   -  Python-Dateien
     
   Beide ZIP-Dateien entpacken in denselben Ordner, in dem auch die Python-Dateien liegen.
